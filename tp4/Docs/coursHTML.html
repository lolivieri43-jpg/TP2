1 - Définition du HTML

HTML signifie HyperText Markup Language. Il permet de structurer le contenu d’une page Web.

⬆ Retour au sommaire

2 - Structure de base d’une page


<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Titre de la page</title>
    </head>

    <body>
        Contenu de la page
    </body>
</html>

La balise <head> contient les informations techniques. La balise <body> contient le contenu visible.
