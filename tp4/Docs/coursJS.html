1 - Structure de base d’un programme

Fichier JavaScript séparé (.js)


/* zone d'en-tête */

/* déclaration des variables globales */
let a;
let b;

/* définition des fonctions */

function maFonction() {
    console.log("Bonjour");
}

/* CORPS DU PROGRAMME */
a = 5;
b = 10;

Il est conseillé de déclarer les variables en début de script pour améliorer la lisibilité.

JavaScript intégré dans une page HTML


<script>
    // Variables globales et fonctions
</script>

<body>
    <script>
        // Instructions
    </script>
</body>

3 - Déclaration des variables

var / let / const


let a = 3;
const PI = 3.14;
var message = "Bonjour";
  • let → variable locale (recommandé)
  • var → variable globale
  • const → constante

Règles de nommage

  • Pas de tiret (-)
  • Pas de mot clé (if, new, typeof…)
  • Sensible à la casse
