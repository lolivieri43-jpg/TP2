1 - Définition du CSS

CSS signifie Cascading Style Sheets. Il permet de styliser le HTML.

⬆ Retour au sommaire

2 - Syntaxe


selecteur {
    propriete: valeur;
}
Exemple : 

p {
    color: blue;
    font-size: 18px;
}
⬆ Retour au sommaire

3 - Les sélecteurs

Par balise


h1 {
    color: red;
}

Par classe


.important {
    color: green;
}

Par identifiant


#titre {
    font-size: 30px;
}

5 - Intégration du CSS

Dans le head

</style>

Fichier externe (recommandé)


<link rel="stylesheet" href="style.css">
