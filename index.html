TP2 – Visual Studio Code

BTS SIO – Découverte de l’environnement de travail

Présentation du projet

Ce projet a pour objectif de découvrir l’environnement de développement Visual Studio Code et les outils associés à la création d’un site web simple.

Outils utilisés

  • Visual Studio Code
  • HTML5
  • Extension Live Preview
  • Git / GitHub

Fonctionnalités mises en œuvre

  • Création d’un projet web
  • Prévisualisation en temps réel
  • Gestion de versions avec Git
  • Publication du projet sur GitHub
