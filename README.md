# TP2 – Environnement de travail VS Code

## Description
Ce projet a été réalisé dans le cadre du TP2 du BTS SIO.
Il a pour objectif de découvrir l’environnement de développement Visual Studio Code
ainsi que la gestion de versions avec Git et GitHub.

## Technologies utilisées
- HTML5
- Visual Studio Code
- Extension Live Preview
- Git / GitHub

## Lancer le projet
Ouvrir le dossier du projet dans Visual Studio Code, puis ouvrir le fichier `index.html`
avec l’extension Live Preview ou directement dans un navigateur web.
